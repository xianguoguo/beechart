柱形图样式设置

1) 柱形方向
    a) 横向
        BarChart(chart).horizontal = true;
        BarChart(chart).skin.statePrinter = new BarChartHorizontalPrinter();
    b) 纵向
        BarChart(chart).horizontal = false;
        BarChart(chart).skin.statePrinter = new BarChartPrinter();

2) 柱状类型
    a) 并排
        dataSet 无 stackGroup 属性即可
    b) 堆叠
        设置 dataSet config 中的 stackGroup 属性
    c) 并排重叠 (暂不支持)

其他设置与折线图相同
